# ANIKIRUN

A jump game using ANIKI at Ikeda, Fukui prefecture.

## Demo
http://codeforfukui.github.io/ANIKIRUN/

## Features
- Tap to make ANIKI jump
- Avoid obstacles (bears) to reach the goal
- Scores and high score displayed

## Usage
1. Open the demo page in a web browser.
2. Tap the screen to start the game and make ANIKI jump.
3. Avoid the obstacles and reach the goal to advance to the next level.
4. The game ends when ANIKI collides with an obstacle.

## Data / API
This project does not use any external data or APIs.
