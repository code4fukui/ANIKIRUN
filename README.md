# ANIKIRUN

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

ANIKI RUN is a simple side-scrolling jump game set in Ikeda, Fukui Prefecture, Japan. You play as Aniki, a character who bounces on a spring, and must jump over running bears to reach the goal.

The game starts with a title screen showing the three main characters: Aniki, a bear obstacle, and a girl who represents the goal. The score and high score are displayed at the top of the screen.

## Demo

You can play the game here: http://codeforfukui.github.io/ANIKIRUN/

## Features

-   Tap the screen to make Aniki jump.
-   Avoid the running bear obstacles to survive.
-   Reach the goal at the end of each stage to advance.
-   Tracks your current score and high score.

## Usage

1.  Open the demo page in a web browser.
2.  Tap the screen to start the game and make Aniki jump.
3.  Avoid the obstacles and reach the goal to advance to the next level.
4.  The game ends when Aniki collides with an obstacle.
